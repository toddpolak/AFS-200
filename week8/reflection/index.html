Discuss your thoughts on this topic.
  • Django is the best framework for web applications, as it allows developers to use modules for faster development. As a developer, you can make use of these modules to create apps, websites from an existing source. It speeds up the development process greatly, as you do not have to code everything from scratch.
  • Surprising to me about Server Development was how easy and convenient the setup process was. You are up and running in little time and there are lots of open-source resources that work with the environment.
  • I enjoy developing both the server (back-end) and the front-end of applications. I like fully understanding how the core of the application works. Once you have a solid backend, you can basically build any front-end around it. Getting a solid backend up and running then testing it with applications like Postman, allow you to feel confident in the foundation of the app so you can move on to solid front-end development with libraries/frameworks like React,Vue, Python and of course, Django!
